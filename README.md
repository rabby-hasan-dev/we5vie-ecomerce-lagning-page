
# We5ive E-Commerce Landing & Product Details Page

This project is a responsive e-commerce landing page and product details page designed using **Next.js** and **Tailwind CSS**, adhering to a Figma design for pixel-perfect accuracy.

## Features

### Landing Page 
- Header with navigation links.
- Featured banners showcasing promotions or offers.
- Categories section for product categorization.
- Product listings with reusable product cards.
- Footer with links and contact information.

### Product Details Page 
- Product information: images, price, description, and reviews.
- Image gallery for product visualization.
- "Add to Cart" button for user interaction.
- Recommendations section displaying similar products.

## Tech Stack
- **Next.js**: React-based framework for server-side rendering and routing.
- **Tailwind CSS**: Utility-first CSS framework for responsive design.
- **Figma**: Design reference for layout, typography, and colors.
